# Coordinate GitHub Repositories v1.0.0

Initial public release of the adaptive repository coordination skill.

## Highlights

- Coordination that begins with context across personal accounts and organizations.
- Equal support for documentation, writing, research, data, operations, software, publishing, and mixed repositories.
- Access diagnosis based on actual agent capabilities rather than host brand.
- Inventory, portfolio review, routing, and cross-repository work patterns.
- Comparative guidance for choosing tools, including the current system and making no change.
- Explicit controls for approval, privacy, untrusted evidence, lifecycle decisions, and recovery.
- Conversation-only fallback plus dated Codex, Claude Code, GitHub Copilot, connector, and MCP guidance.
- Standalone Agent Skill, Codex plugin, and Claude plugin release packages.

## Installation

Use the standalone ZIP for the most portable global installation. See root `INSTALL.md` in the repository for host locations and validation.
