# Coordinate GitHub Repositories v1.0.0

Initial public release of the adaptive repository coordination skill.

## Highlights

- Context-first coordination across personal accounts and organizations.
- First-class support for documentation, writing, research, data, operations,
  software, publishing, and mixed repositories.
- Access diagnosis based on actual agent capabilities rather than host brand.
- Inventory, portfolio review, routing, and cross-repository work patterns.
- Comparative tool-fit guidance with current-system and no-change options.
- Explicit approval, privacy, untrusted-evidence, lifecycle, and recovery gates.
- Conversation-only fallback plus dated Codex, Claude Code, GitHub Copilot,
  connector, and MCP guidance.
- Standalone Agent Skill, Codex plugin, and Claude plugin release packages.

## Installation

Use the standalone ZIP for the most portable global installation. See
`docs/INSTALL.md` in the repository for host-specific locations and validation.
